# Book Reports



These are **original study reports** for your frontend review page and teacher marking.



## How to use in a frontend



1. Load `catalog.json` to list families and reports.

2. For each report, fetch `report.json` (structured) or `report.md` (readable text).

3. Show **Summary** then **Chapters** - author bios and licenses were removed on purpose.
