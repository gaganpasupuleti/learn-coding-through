#### Chapter 5: Applied: Canvas App Controls and Formulas *(Level: Beginner)*

**Chapter focus: Canvas App Controls and Formulas** *(Level: Beginner)*

// Patch(Inspections, Defaults(Inspections), {DeviceId: galDevices.Selected.ID})
